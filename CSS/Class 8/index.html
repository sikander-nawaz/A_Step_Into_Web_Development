CSS Foundations

This page demonstrates the three ways to apply CSS to an HTML document.

Internal CSS

Styles written inside a <style> tag in the <head> section. Applies to the current page only.

External CSS

Styles written in a separate .css file and linked using a <link> tag. Reusable across multiple pages.

Inline CSS

Styles applied directly on an element using the style attribute. Highest specificity, but hardest to maintain.
